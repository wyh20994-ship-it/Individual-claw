# 高德地图生活服务

## 使用说明

| 命令 | 功能 |
|------|------|
| `/weather <城市>` | 查询城市天气（当前 + 3天预报） |
| `/map search <关键词> <城市>` | POI 搜索（餐厅、酒店等） |
| `/map route <起点> <终点>` | 驾车路线规划 |

也可以用自然语言触发，例如：
- "北京今天天气怎么样"
- "附近有什么好吃的"

## 执行逻辑

1. 解析意图：天气 / POI搜索 / 路线
2. 调用高德 Web API（需要 `AMAP_API_KEY`）
3. 格式化返回结果

## 依赖
- 高德开放平台 Web 服务 API Key
